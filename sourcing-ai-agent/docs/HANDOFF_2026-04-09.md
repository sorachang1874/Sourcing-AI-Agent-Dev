## Harvest Lessons Now Considered Canonical

### 1. Large-org current roster must be probe-driven

Anthropic 的美国 current roster root probe 已验证：

- `United States root = 2837`
- provider capped

当前已经稳定收敛的 live shard set：

- `United States / Engineering = 1098`
- `United States / Remaining after Engineering = 1928`

这意味着：

- 不再默认写死 `US + Engineering / exclude Engineering`
- 先 probe root
- 只有当 root 超 cap，才继续 split
- 只执行 live 估算已落到 cap 以内的 shard
